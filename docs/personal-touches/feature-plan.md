# Feature Plan: Personal Touches

## Overview
Personal Touches add character and "soul" to the Tiny Tamagotchi through easter eggs and quirky reactions that aren't strictly tied to survival stats.

## User Flow
- **Interaction Reaction**: Clicking on the pet's sprite triggers a random "poke" reaction.
- **Night Mode**: The UI background automatically darkens if the current local time is between 10 PM and 6 AM.
- **Easter Egg**: Naming the pet a specific "Dev Name" (e.g., "Gemini" or "Antigravity") unlocks a unique accessory or color.
- **Idle Behavior**: If the user doesn't interact for 30 seconds, the pet performs a random idle animation (e.g., spinning or waving).

## Success Criteria
- Easter eggs are discoverable but not required for gameplay.
- Night mode reflects real-world time accurately.
- Idle behaviors add life without interrupting core gameplay actions.
- The pet feels less like a set of numbers and more like a companion.
